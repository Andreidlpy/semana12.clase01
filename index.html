Como se transfiere la informacion basado en el protocolo http.

El usuario inicia una solicitud al servidor, el servidor procesa esa solicitud y genera una respuesta que sera enviada al usuario.

Que son los verbos HTTP y cuales usamos.

son acciones que se utilizan en las solicitudes HTTP

  • POST: enviar información
  • GET: obtener información
  • PUT: actualizar información
  • DELETE: actualizar información

Que es un crud.

Un CRUD es un acrónimo que se refiere a las cuatro operaciones básicas que se pueden realizar en una base de datos o en una aplicación que gestiona datos.

  • C: Create
  • R: Read
  • U: Update
  • D: Delete

Que es asincronia y sincronia.

La sincronia es cuando las tareas o procesos se ejecutan uno después del otro en un orden

La asincronia es cuando las tareas pueden iniciarse y ejecutarse en paralelo o en momentos diferentes, y no es necesario esperar a que una tarea se complete antes de comenzar otra.

Como maneja la asincronia js.

JavaScript maneja la asincronía a través de un modelo de ejecución basado en eventos y callbacks, aunque en versiones más recientes del lenguaje, se han introducido características como Promesas y async/await para simplificar el manejo de código asíncrono.
